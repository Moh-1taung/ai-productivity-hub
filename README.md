A workspace that maximizes productivity and saves time spent on research, meetings, and emails. Application is built with lovable.dev for CAPACITI.
https://lovable.dev/projects/29da5289-05a3-42a9-a7a8-67bef87923b8
The project consists of the following features:
AI Research Assistant allows the user to enter any topic and get a summary and 3 key insights of the topic. Perfect for users who need immediate research results, varying from students to workplace research, as well as curious individuals.
Meeting Notes Summarizer, the user just has to paste their meeting transcript to allow the application to summarise the notes in 3 bullet points and set apart key decisions as well as action items that appear from the transcript.
AI Email Generator, by just telling the bot what the email is about, what tone to use, and who it concerns. The application can immediately write the email in the preferred tone(friendly/professional/formal) and directed to the correct audience( team/ client/manager). The user can copy the draft to their official email application. In the 
AI Task Planner, the user is able to list their task on the application and set them in prioritization order, which includes dates.
AI Chatbot allows for an immediate interaction with the AI. The user can ask questions that are both formal and casual.
Frontend: React + Tailwind + Typescript
Built with: Lovable.dev
